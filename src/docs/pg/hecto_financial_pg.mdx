# PG 결제 서비스

헥토파이낸셜에서 제공하는 신용카드 결제 서비스입니다.

## 주요 특징

- **다양한 카드사 지원**: 국내외 주요 카드사 지원
- **안전한 결제**: PCI DSS 준수 보안 시스템
- **실시간 승인**: 빠른 결제 승인 처리
- **간편한 연동**: 표준 API를 통한 쉬운 연동

## 지원 결제수단

### 신용카드
- 국민카드, 신한카드, 삼성카드, 현대카드
- 하나카드, 우리카드, 롯데카드
- BC카드, NH농협카드

### 간편결제
- 삼성페이, 카카오페이, 네이버페이
- 페이코, 토스

## API 개요

### 결제 API
- **결제 요청**: 결제창 호출 및 결제 처리
- **결제 승인**: 결제 승인 처리
- **결제 취소**: 전체/부분 취소 처리

### 조회 API
- **거래 조회**: 결제 내역 조회
- **정산 조회**: 정산 내역 조회

## 연동 프로세스

1. **결제 요청**: 가맹점에서 결제 요청
2. **결제창 호출**: PG 결제창 표시
3. **카드 정보 입력**: 고객이 카드 정보 입력
4. **결제 승인**: 카드사 승인 처리
5. **결과 통보**: 결제 결과 가맹점에 통보

## 보안

### 개인정보 보호
- **암호화**: AES-256 암호화
- **토큰화**: 카드번호 토큰화 처리
- **SSL/TLS**: 전송 구간 암호화

### 인증
- **3D Secure**: 추가 인증 지원
- **ISP/안심클릭**: 국내 인증 서비스
- **간편인증**: 생체인증, PIN 인증

## 수수료

결제 수수료는 카드사 및 가맹점 조건에 따라 차등 적용됩니다.
